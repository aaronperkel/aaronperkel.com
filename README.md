# [Personal Web Site](https://www.aaronperkel.com)

## Description
This is my personal website that I use to display some projects that I work on, a little info about myself, and my resume. Originally built in Notion, then hand-written in HTML/PHP/JS, and now a [Next.js](https://nextjs.org) + [Tailwind CSS](https://tailwindcss.com) app deployed on Vercel.

## Development

```bash
npm install
npm run dev    # http://localhost:3000
```
